    整合包注意事项 ⚠

    1. 使用整合包前请确保你阅读了服务器条例

    2. 整合包禁止对外分享，一经发现，将取消白名单资格。

    3. 整合包中的部分内容拥有保密协议，不允许分享。一经发现，将取消白名单资格。

    4. 整合包中的部分内容被独立修改，不允许分享。一经发现，将取消白名单资格。

    5. 整合包仅用于连接服务器使用，不做其它用途。

    解压问题

    1. 如遇到解压失败或解压不成功，请检查文件后缀是否为压缩包格式。

    2. 解压之后的整合包为可以被直接启动，不需要进行其他配置。

    3. 解压后的整合包未提供玩家信息，请注册填写玩家ID。

    4. 压缩包被密码加密，请先阅读服务器条例后进群获取

    启动问题

    1. 启动前请查看您的启动器配置是否正确。

    2. 第一次启动游戏较慢，请耐心等待。

    3. 启动器需要配置合适Java版本，启动前请检查Java是否配置完成。

    4. 服务器需要8GB启动器内存，请确保内存充足的情况下进入服务器

    5. 如遇到启动奔溃问题请移至 崩溃问题

    配置问题

    1. 服务器推荐配置以下--------------------------------------------

    处理器：intel i5 7代 或 AMD 锐龙5 及以上

    显卡：GTX 1050ti 或 AMD R9 380 及以上 4GB显示内存

    内存：16GB及以上

    硬盘：至少2GB可用空间

    2. windows 64位 Java版本需要64位

    3. 服务器可用配置以下--------------------------------------------

    处理器：intel i3 5代 或 AMD 速龙x4 及以上

    显卡：GTX 550ti 或 AMD R7 240 及以上 1GB显示内存

    内存：8GB及以上

    硬盘：至少2GB可用空间

    奔溃问题

    1. 客户端启动时奔溃，请检查安装的mod是否兼容，内存是否充足（8GB）

    2. 客户端启动完成在主界面奔溃，请检查mod是否兼容，检查mod需要的前置。

    3. 连接服务器时奔溃，请检查可用内存是否充足（8GB），检查是否安装了非服务器所选mod。

    4. 游戏中奔溃，请移至 游戏内问题

    5. 游戏启动就奔溃，请检查启动器配置是否正确，java版本是否正确，如还无法解决，请联系腐竹。
